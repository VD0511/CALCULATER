# 📱 Modern Casio-Style Calculator

Ek clean aur fully functional web calculator jo **HTML, CSS, aur JavaScript** se bana hai. Ismein basic calculations ke saath-saath modern UI aur keyboard support bhi diya gaya hai.

## ✨ Key Features
- **Modern UI:** Dark mode design aur responsive layout (CSS Grid & Flexbox).
- **Basic Operations:** Addition, Subtraction, Multiplication, aur Division.
- **Keyboard Support:** Laptop ke keyboard se direct typing (Numbers, Enter, Backspace, Esc).
- **Error Handling:** Invalid calculations hone par "Error" message show karta hai.
- **Real Casio Feel:** "A.C" (All Clear) aur "C" (Backspace/Delete) functions.

## 🛠️ Tech Stack Used
- **HTML5:** Structure banane ke liye.
- **CSS3:** Styling, Animations, aur Responsive Grid layout ke liye.
- **JavaScript (ES6):** Calculation logic aur DOM manipulation ke liye.

## 🚀 How to Use
1. Project ko clone karein ya download karein.
2. `index.html` file ko kisi bhi browser mein open karein.
3. Mouse se click karein ya keyboard ka use karke apni calculation karein!
